Dia Mundial da Conscientização sobre Acessibilidade

No dia 21 de Maio se comemora o Dia Mundial da Conscientização sobre Acessibilidade, em que se relembra a luta histórica para o reconhecimento das necessidades específicas.

O que é acessibilidade digital

Acessibilidade Digital é um conjunto de medidas que devem ser realizadas para garantir que todas as pessoas possam acessar, navegar e interagir com a web de forma autônoma

Inclusão digital e web para todos

O Movimento Web para Todos tem como objetivo mobilizar a causa da acessibilidade digital

Importância de Diretrizes WCAG

As diretrizes WCAG são o conjunto de normas, feitas pela W3C para acessibilização de sites na internet. Seu uso é de extrema importância para a padronização dos sites de forma que, para além de uma conscistência de uso, tecnologias assistivas, como leitores de tela, sejam eficientes.

Estrutura acessível em HTML e CSS

Para que os sites sejam acessíveis, são disponibilizadas pelas diretrizes, como a WCAG, estruturas de composição de sites pelo HTML e CSS. Por exemplo, deve-se utilizar as seguintes tags semânticas:

  • header: cabeçalho
  • nav: barras de navegação
  • main: conteúdo principal
  • section: seções do texto
  • aside: menus laterais
  • footer: rodapé
Além disso, deve-se usar a seguinte hierarquia de títulos:
  1. h1
  2. h2
  3. h3
  4. h4
  5. h5
  6. h6
Embora seja recomendado utilizar somente até o terceiro título.

Boas práticas de acessibilidade digital

Contraste de cores e legibilidade

Um aspecto importante para a acessibilidade digital é fazer com que haja contraste entre elementos relevantes. Por exemplo, textos devem estar em cores e tons opostos ao fundo. Isso é importante para pessoas com baixa visão e para uma navegação com clareza para pessoas

Importância da descrição de imagens

Também é importatnte escrever a audiodescrição das imagens através do texto alt. Dessa forma, pessoas que usam leitores de tela podem compreender o uso das imagens no site.
